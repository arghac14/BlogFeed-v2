# BlogFeed-v2
A multi-user blogging application built using MERN (MongoDB, Express, React, Node) stack with cloud-based image and video management service cloudinary.

# To run:

```
$git clone https://github.com/arghac14/BlogFeed-v2.git
$cd blogfeed
$npm install
$npm start

$cd ..
$cd server
$npm install
$npm start

```
